# Audio Assets

Place the following audio files in this directory before building:

## Background Music
- `bgm_casino.mp3` — Looping casino ambiance (plays across the whole app)

## Sound Effects
- `sfx_click.mp3` — Subtle button/navigation click
- `sfx_deal.mp3`  — Card dealing sound
- `sfx_win.mp3`   — Win result fanfare
- `sfx_lose.mp3`  — Lose result tone

All files are MP3. The app handles missing files gracefully (silent fail).
